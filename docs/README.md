# Welcome

## Why use Vue?

Vue is a frontend framework that
- is easy-to-learn
- has a strong community
- makes it easy to create reusable components that include
    - template markup
    - logic/script
    - styles that can be scoped
- makes it easy to create polished UIs with Vuetify
